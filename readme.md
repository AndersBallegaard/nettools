# Nettools
### A docker container full of tools for network automation and troubleshooting

## Included tools
* Python3
* Ansible
* Curl
* Wget
* net-tools (ifconfig)
* iproute2 (ip commands)
* Ping
* Traceroute
* nmap
* tcpdump
* iperf3
* netmiko
* requests
* ncclient

If you want more tools create an issue or a pull request
